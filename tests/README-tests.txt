This directory contains tests for the catalogue.

Run the code with the filenames of the main source catalogue, the
component catalogue, the artefact catalogue and the source list.
e.g. in blend directory:

python $LGZPATH/tests/testsuite.py merge_out.fits merge_comp_out.fits merge_art_out.fits LOFAR_HBA_T1_DR1_catalog_v0.9.srl.fixed.fits
